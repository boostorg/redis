◆ ignore

using aedis::ignore = typedef adapter::detail::ignore

Tag used to ignore responses.

For example

std::tuple<aedis::ignore, std::string, aedis::ignore> resp;

will cause only the second tuple type to be parsed, the others will be ignored.
