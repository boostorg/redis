Detailed Description

template<class Storage, class Command>
class aedis::resp3::serializer< Storage, Command >

Creates a Redis request from user data.

A request is composed of one or more redis commands and is referred to in the redis documentation as a pipeline, see https://redis.io/topics/pipelining.

For example

std::string request;
auto sr = make_serializer(request);
sr.push(command::hello, 3);
sr.push(command::flushall);
sr.push(command::ping);
sr.push(command::incr, "key");
sr.push(command::quit);
co_await async_write(socket, buffer(request));

Appends a new command to the end of the request.

For example

std::string request;
auto sr = make_serializer<command>(request);
sr.push(command::set, "key", "some string", "EX", "2");

will add the set command with payload "some string" and an expiration of 2 seconds.
