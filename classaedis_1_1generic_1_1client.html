Detailed Description

template<class AsyncReadWriteStream, class Command>
class aedis::generic::client< AsyncReadWriteStream, Command >

A high level Redis client.

This class represents a connection to the Redis server. Some of its most important features are

  1. Automatic management of commands. The implementation will send commands and read responses automatically for the user.
  2. Memory reuse. Dynamic memory allocations will decrease with time.

For more details, please see the documentation of each individual function.

Starts communication with the Redis server asynchronously.

This class performs the following steps

  • Connect to the endpoint passed in the function parameter.
  • Start the async read operation that keeps reading responses to commands and server pushes.
  • Start the async write operation that keeps sending commands to Redis.
